# Assign a Layout

This function allows user to assign different layouts to each different Products, Collections, Blogs or Articles.

Example for Product template. Go to App, switch to **Layouts assigned** tab **>** Click **Assign layout** button.

![](../../../.gitbook/assets/image%20\(383\).png)

Search and Select **one product** in your store.

![](../../../.gitbook/assets/image%20\(273\).png)

Select a **Product page Template/Layout** that you have created and customized > Click **Apply** button.

{% hint style="info" %}
Note: This layout is only applied to the only one product selected. To the products that are not assigned a certain Template, they use the Layout that has been making default.
{% endhint %}
