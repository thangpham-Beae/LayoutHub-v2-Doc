# LayoutHub Dashboard

## Overview

In this series of articles, we will introduce you to the LayoutHub Dashboard interface and the functions available in LayoutHub Dashboard.

## What is LayoutHub dashboard?

LayoutHub dashboard is your main control center in LayoutHub, providing quick access to page creation, management tools, app settings, and enhanced functionality.

**The LayoutHub Dashboard is organized into two main sections:**&#x20;

1. Sidebar Navigation.&#x20;
2. Main Workspace.

{% hint style="success" %}
Tip: In the main workspace, there is always a chat box so you can contact the LayoutHub Customer Success team. Available 24/7 to answer all your questions.&#x20;
{% endhint %}
