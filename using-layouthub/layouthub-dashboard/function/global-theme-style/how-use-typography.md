# How use Typography?

## Overview

In this article we will introduce the Typography feature in Global theme style and how to use it to set up consistent text styling in the theme.

## What is typography?

Typography brings together presets of 8 typography types in the template including:&#x20;

* Heading 1
* Heading 2
* Heading 3
* Heading 4
* Heading 5
* Body text&#x20;
* Body text 1
* Body text 2

These presets will help you reuse typography without having to set it up from scratch.

\[ Screenshot ]

## How to customize typography?

Click on typography item to open its editing panel, then adjust the available text styling options within the preset.

\[ Screenshot ]

## Frequently Asked Questions

### How can I change typography when I edit my page?

You can adjust typography directly through the Heading Size or Text Size options in the text block.

\[ Screenshot ]

### I can't find the font family I need?

Font family are provided by Google Fonts. If you want to add a custom font that is not available in Google Fonts, please contact our support team for assistance.&#x20;

### Can I set typography for different devices?

Yes, typography can **work independently on Desktop, Tablet and Mobile**.

\[ Screenshot ]

We hope this article helps you understand how to use Typography feature in Global Theme style to customize typography presets and set consistent typography for your theme.
