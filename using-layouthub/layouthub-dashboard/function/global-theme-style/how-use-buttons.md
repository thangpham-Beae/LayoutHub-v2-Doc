# How use Buttons?

## Overview

In this article we will guide you how to customize buttons style preset so that they can be reused and consistency across pages.

## What are Buttons?

Buttons are a set of reusable button presets included in Global Theme Style. They allow you to apply consistent button designs across all pages.\
There are three preset types available: Primary, Secondary, and Tertiary buttons.
