# How to use Variant Picker?

## Overview

In this article, we'll show you how to use Variant Picker to display your product variant.

## What is Variant Picker?

Variant picker (lựa chọn biến thể) trên cửa hàng thương mại điện tử đóng vai trò quan trọng trong việc nâng cao trải nghiệm mua sắm của khách hàng bằng cách cho phép người mua xem và chọn các phiên bản khác nhau của sản phẩm một cách liền mạch, chẳng hạn như kích thước, màu sắc, chất liệu hoặc các thuộc tính khác.

Variant Picker thường xuất hiện ở Feature Collection và Feature Product.&#x20;

## How to use Variant Picker?

In this section we will guide you through the setup, meaning and usage of each option in Variant Picker.

### Show Variant

&#x20;Click option **Show variant** to display variant picker.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-11-21 at 15.04.46.png" alt=""><figcaption></figcaption></figure>

### Variant name

Enter the name of the variant that you want to display as a color watch.&#x20;

{% hint style="info" %}
Where can I get the variant name? Check [this article](../../../faq-and-common-issues/faq/where-can-i-get-the-variant-name.md).
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2025-11-21 at 15.48.53.png" alt=""><figcaption></figcaption></figure>

{% hint style="warning" %}
To ensure the LayoutHub Variant Picker detects your variant correctly, please enter the variant name exactly as it appears — including proper uppercase and lowercase letters.
{% endhint %}

### Show variant image

You can optionally display Variant color or display Variant image.&#x20;

If the show variant image option is not enabled, the variant image will be displayed by default.

#### Variant color



What is the difference between variant color and variant image?

* **Variant Color**:\
  Displays variant options as **color swatches** (for example: red, blue, black).\
  You only need to enter the **color name** or **color code**.\
  Best when you want a simple color representation.
* **Variant Image**:\
  Displays each variant using an actual **image**.\
  Every variant has **its own photo**.\
  Best when customers need to see how each option looks in real life.
