# Export & Import

Save more time with Import and Export feature in LayoutHub. It's easier for reserving a custom-made design offline and continuing to have some revision online.

Let's look at the bottom left corner of the Editor, it's the **Page Actions** button.

![](/.gitbook/assets/import.png)

## 1 - Export current page

* Click **Export page to file** > Enter **a name** for the exported file.

![](/.gitbook/assets/import1.png)

* Save the file under **.hub** format.

![](/.gitbook/assets/import2.png)

## **2 - Import your page**

Now that you know how to export, you may want to know in which way to import as well.

Please open your under-struction **Template** > **Choose page actions >** Click **Import file \*.hub**

* Select **the exported file \*.hub > Open**

![](/.gitbook/assets/import3.png)

_**Result:**_ The New template has the settings of the Old template.

## 3 - Clear all sections

* Go to **Page actions > Choose Clear all sections**

![](/.gitbook/assets/image%20(363).png)

* All the sections and blocks are cleared and you start to build your page again.

![](/.gitbook/assets/image%20(352).png)
